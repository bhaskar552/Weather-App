# Weather-App
User Friendly Weather App with Chart

## Run Locally

Clone the project

```bash
  git clone https://github.com/bhaskar552/Weather-App
```

Go to the root directory

```bash
  cd Weather-App
```

Install dependencies

```bash
  npm install --force
```

Start the Project

```bash
  npm start
```
Live Demo -: https://react-weather-chart.netlify.app/
